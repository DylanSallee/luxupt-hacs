{# ===== BUTTON USAGE RULES ===== #}

Button Usage Rules

DO

  • • primary for main action (Save, Submit, Create)
  • • secondary for secondary actions (View, Settings, Edit)
  • • ghost for Cancel, Close, Back (de-emphasized)
  • • success for Connect, Enable, Approve
  • • danger for Delete, Remove, Disconnect

DON'T

  • • Multiple primary buttons in the same row
  • • Danger button for Cancel (use ghost)
  • • Gray buttons for clickable actions (gray = disabled)
  • • Custom colors - use the semantic variants
  • • Blue buttons - primary is magenta in this theme
