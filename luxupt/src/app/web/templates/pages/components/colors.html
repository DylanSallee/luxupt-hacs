{# ===== TEXT HIERARCHY ===== #}

Text Hierarchy (2 levels only)

text-white Headings, important labels, card titles
text-surface-200 Everything else - body text, metadata, descriptions, labels

Keep it simple: white for emphasis, surface-200 for everything else

No surface-300 or surface-400 for text. If it needs to be read, it needs to be visible.

{# ===== LINKS & INTERACTIVE TEXT ===== #}

Links & Interactive Text

This is a link text-primary-400 hover:text-primary-300
text-primary-400 hover:text-primary-300

All clickable text uses primary color, not gray or white
