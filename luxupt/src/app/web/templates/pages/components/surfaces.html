{# ========================================================================= SURFACES & BACKGROUNDS - Layer hierarchy for dark UI ========================================================================= #}

Surfaces & Backgrounds

MANDATORY — Use these Tailwind classes. Never use inline rgba().

{# ===== LAYER HIERARCHY ===== #}

Layer Hierarchy (z-depth)

Page

bg-surface-900

Page background

Card

bg-surface-500/50

Cards, containers

Nested

bg-surface-800/50

Sections within cards

Overlay

bg-surface-900/50

Modals, panels + backdrop-blur

{# ===== WHEN TO USE EACH ===== #}

When To Use Each Class

Class Use For Examples
bg-surface-900 Page background, inputs Set in body, form inputs
bg-surface-500/50 Primary containers Cards, panels, stat boxes
bg-surface-800/50 Nested sections Section cards, info boxes
bg-surface-700/50 Interactive elements Tags, interval pills, hover states
bg-surface-900/50 backdrop-blur Overlays with blur Modal backdrops, panel overlays
{# ===== BORDER RADIUS STANDARD ===== #}

Border Radius Standard

rounded-xl

Cards, containers

rounded-lg

Buttons, inputs, nested

rounded-md

Small elements

rounded-full

Pills, avatars, dots

RULE: Use rounded-xl for card containers, rounded-lg for buttons, inputs, and nested elements.

{# ===== BORDERS ===== #}

Border Classes

border-surface-400/30 Standard card/container border
border-surface-400/20 Subtle nested element border
border-surface-400/50 Emphasized border (panels, sections)

{# ===== WHAT NOT TO DO ===== #}

NEVER Do This

✗
style="background: rgba(40, 40, 40, 0.5)"

Use bg-surface-500/50 instead

✗
bg-gray-800

Use bg-surface-800 - we use surface, not gray

✗
rounded-lg for cards

Use rounded-xl for cards/containers
